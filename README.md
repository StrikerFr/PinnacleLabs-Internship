### 🌦️ WeatherWatch AI
WeatherWatch AI is a real-time weather intelligence platform designed as an immersive experience rather than a traditional dashboard.

Features include:
* **Real-time forecasts:** Scraping local metrics using Open-Meteo services.
* **Air quality insights:** Telemetry of ozone, nitrogen dioxide, PM2.5, and PM10 particles.
* **Disaster alerts:** Real-time global hazard tags sourced from GDACS.
* **Weather-related news:** Relevant environment articles.
* **Travel recommendations:** AI-computed transit guidelines.
* **Health suggestions:** UV and air index warning thresholds.
* **AI-powered weather assistant:** Natural language narration utilizing Groq Llama-3.3-70b processing.

* **Branch:** `Task3-AryanGarg`
* **Source Link:** [WeatherWatch AI Branch](https://github.com/StrikerFr/PinnacleLabs-Internship/tree/Task3-AryanGarg)
